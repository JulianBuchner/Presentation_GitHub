GitHub - Funktionen

branch

unabhängige Entwicklungszweige

betreffen nicht Hauptprojekt

checkout

zwischen branches wechseln

commit

Änderungen lokal speichern

Rollbacks möglich

-a ➔ Alle Änderungen

-m ➔ Commitnachricht

-amend ➔ ändert letzen commit

push

Änderungen in das Repo auf aktuellen branch speichern

merge

Änderungen verschiedener branches zu einem migrieren

pull

Änderungen vom Repo laden

reset

Branch auf bestimmten commit zurücksetzen

revert

Neuer Commit setzt vorherigen Commit zurück

rebase

Branchverlauf auf anderen Punkt setzen

cherry-pick

Commits auf andere Branches übertragen

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.on( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Take a Moment

Press B or . on your keyboard to pause the presentation. This is helpful when you're on stage and want to take distracting slides off the screen.
