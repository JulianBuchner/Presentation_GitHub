GitHub - Funktionen

branch

unabhängige Entwicklungszweige

betreffen nicht Hauptprojekt

checkout

zwischen branches wechseln

commit

Änderungen lokal speichern

Rollbacks möglich

-a ➔ Alle Änderungen

-m ➔ Commitnachricht

-amend ➔ ändert letzen commit

push

Änderungen in das Repo auf aktuellen branch speichern

merge

Änderungen verschiedener branches zu einem migrieren

pull

Änderungen vom Repo laden

reset

Branch auf bestimmten commit zurücksetzen

revert

Neuer Commit setzt vorherigen Commit zurück

rebase

Branchverlauf auf anderen Punkt setzen

cherry-pick

Commits auf andere Branches übertragen
